Mongo Movie

Ce projet est une application web développée en PHP et MongoDB permettant de gérer une bibliothèque de films. Elle permet aux utilisateurs d’ajouter, modifier, supprimer et interroger des informations sur les films, les acteurs et les réalisateurs.

Le système utilise une base de données MongoDB pour stocker et récupérer les données, offrant un exemple concret de gestion de données liées au cinéma avec NoSQL.

Fonctionnalités :

  • Créer, Lire, Modifier et Supprimer des films avec MongoDB
  • Stockage et récupération des détails des films (titre, genre, date de sortie, etc.)
  • Recherche et filtrage par acteurs, films et réalisateurs

Technologies utilisées :

  • PHP : langage principal pour le backend
  • MongoDB : base de données pour gérer les informations liées aux films
  • JavaScript : pour les interactions côté client
  • CSS : pour le style de l’application
