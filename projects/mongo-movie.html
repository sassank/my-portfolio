Mongo Movie

This project is a web application built with PHP and MongoDB that manages a movie library. It allows users to add, update, delete, and query information about movies, actors, and directors. The system interacts with a MongoDB database to store and retrieve data, providing a practical example of managing movie-related data using NoSQL.

Features:

  • Create, Read, Update & Delete movie datas with MongoDB.
  • Storing and retrieving movie details (e.g., title, genre, release date).
  • Search and filter movies by actors, films and directors.

Technologies Used:

  • PHP: Main programming language for backend functionality.
  • MongoDB: Used as the database to manage movie-related data.
  • JavaScript: Used for client-side interactions
  • CSS: For styling the project.
