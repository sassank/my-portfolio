ViteDrive - Plateforme de location de véhicules

ViteDrive est une plateforme web permettant aux utilisateurs de rechercher, réserver et gérer la location de véhicules en ligne. L’application offre une interface fluide et moderne, ainsi qu’un espace d’administration pour la gestion des véhicules, des utilisateurs et des réservations.

Fonctionnalités principales :

  • Recherche de véhicules disponibles selon les dates et le modèle.
  • Création et gestion de compte utilisateur.
  • Réservation en ligne.
  • Interface administrateur pour gérer les véhicules et les réservations.
  • Affichage dynamique et responsive du catalogue de véhicules.

Technologies utilisées :

  • Front-end : React.js, Axios
  • Back-end : Spring Boot (Java), REST API
  • Base de données : PostgreSQL
  • Outils : Git, GitHub, Postman
