ownCloud server

This project focuses on setting up an ownCloud server on a Debian operating system. ownCloud is a powerful open-source platform that provides file synchronization, sharing, and collaboration services. The goal is to configure a fully functional ownCloud server, allowing users to securely store and manage their files, with support for LDAP integration and access controls.

Features:

  • Installation of ownCloud on a Debian Buster machine with specific configurations.
  • Setup of network interfaces, including NAT and static IP addresses.
  • Database creation and user configuration for ownCloud.
  • Customization of Apache settings to simplify the URL structure and ensure seamless access to the ownCloud platform.

Technologies Used:

  • A virtual machine of Debian Buster.
  • ownCloud: The platform used for cloud storage and file management.
  • Apache: Web server used to serve the ownCloud instance.
  • MySQL: Database system for managing ownCloud's data.
  • LDAP: Lightweight Directory Access Protocol for integrating Active Directory authentication.
