Serveur ownCloud

Ce projet consiste à mettre en place un serveur ownCloud sur un système d’exploitation Debian. ownCloud est une solution open-source puissante qui permet la synchronisation, le partage et la gestion sécurisée de fichiers. L’objectif est de configurer un serveur entièrement fonctionnel, permettant aux utilisateurs de stocker et gérer leurs fichiers en toute sécurité, avec la prise en charge de l’intégration LDAP et des contrôles d’accès.

Fonctionnalités principales :

  • Installation d’ownCloud sur une machine Debian Buster avec une configuration spécifique.
  • Configuration des interfaces réseau, incluant le NAT et les adresses IP statiques.
  • Création de la base de données et configuration des utilisateurs pour ownCloud.
  • Personnalisation d’Apache afin de simplifier la structure des URL et d’assurer un accès fluide à la plateforme.

Technologies utilisées :

  • Système d’exploitation : Debian Buster (machine virtuelle)
  • Plateforme : ownCloud – solution de stockage et gestion de fichiers cloud
  • Serveur web : Apache
  • Base de données : MySQL
  • Annuaire : LDAP (intégration avec Active Directory)
